# FEUP-FPRO

Exercises from the FPRO course unit at FEUP.

Exercise descriptions and answers taken from the [unit website](https://fpro.fe.up.pt) automatically. Formatting won't be perfect.

## Folder structure

* **ME\***: Practice exams
* **PE\***: Practical exams
* **Py\***: Weekly practice exercises
* **RE\***: Weekly exercises
